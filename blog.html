Blog: Exploring Data Analysis with R

Exploratory Data Analysis 1: Scatter Plot

Introduction

In this EDA, we'll create a scatter plot to explore the relationship between two numeric variables.

R Code


library(ggplot2)

# Set seed for reproducibility
set.seed(123)

# Generate a random dataset with two numeric variables and one categorical variable
n <- 100  # Number of observations
data <- data.frame(
  X1 = rnorm(n),
  X2 = rnorm(n, mean = 2),
  Category = sample(c("A", "B"), n, replace = TRUE)
)

# Scatter plot
scatter_plot <- ggplot(data, aes(x = X1, y = X2, color = Category)) +
  geom_point() +
  labs(title = "Scatter Plot", x = "X1", y = "X2")

# Save the scatter plot
ggsave("scatter_plot.png", plot = scatter_plot)

Exploratory Data Analysis 2: Histogram

Introduction

In this EDA, we'll create a histogram to explore the distribution of a numeric variable.

R Code


# Histogram
histogram_plot <- ggplot(data, aes(x = X1, fill = Category)) +
  geom_histogram(binwidth = 0.5, position = "identity", alpha = 0.7) +
  labs(title = "Histogram", x = "X1", y = "Frequency")

# Save the histogram plot
ggsave("histogram_plot.png", plot = histogram_plot)

Exploratory Data Analysis 3: Boxplot

Introduction

In this EDA, we'll create a boxplot to explore the distribution of numeric variables across categories.

R Code


# Boxplot
boxplot <- ggplot(data, aes(x = Category, y = X2, fill = Category)) +
  geom_boxplot() +
  labs(title = "Boxplot", x = "Category", y = "X2")

# Save the boxplot
ggsave("boxplot.png", plot = boxplot)
